Run a dozen AI agents. Keep track of all of them.

Clawterm shows you which agent is waiting, which one errored, and which build just finished — so you stop checking tabs and start shipping.

Download for macOS View on GitHub

Built for the multi-agent workflow

|

See everything at a glance

Vertical tabs show each session's state — idle, running, agent waiting, error — without clicking through them.

#

Split and conquer

Horizontal and vertical splits let you run multiple agents side by side within a single tab.

>_

Get notified, not interrupted

Desktop notifications when agents need input or long commands finish. Focus on what matters.

~

Native and fast

Rust PTY backend, WebGL rendering, no Electron. Transparent window, traffic lights, macOS native feel.

!

Stay organized

Pin tabs, mute noisy ones, drag to reorder, double-click to rename. Command palette for everything else.

^

Session persistence

Tabs and working directories restore on relaunch. Auto-updates install with one click from the sidebar.

Install

curl -fsSL https://raw.githubusercontent.com/clawterm/clawterm/main/install.sh | bash

Or download the .dmg from GitHub Releases. macOS (Apple Silicon & Intel).
