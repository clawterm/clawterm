Documentation

Everything you need to use and configure Clawterm.

Installation

macOS

curl -fsSL https://raw.githubusercontent.com/clawterm/clawterm/main/install.sh | bash

Or download the .dmg from GitHub Releases.

After installing from the DMG, you may need to run:

xattr -cr /Applications/Clawterm.app

This clears the macOS quarantine flag. It won't be needed once Apple Developer Program approval is complete.

Windows

Download the .exe installer from GitHub Releases and run it. Requires Windows 10 (1809+) or later.

Auto-updates

Clawterm checks for updates automatically. When a new version is available, click Update in the sidebar.

Configuration

Config is stored at:

  • macOS: ~/.config/clawterm/config.json
  • Windows: %APPDATA%\clawterm\config.json

Created with defaults on first launch. Only include keys you want to override.

{
  "shell": "/bin/zsh",
  "font": { "family": "Menlo, Monaco, monospace", "size": 14 },
  "cursor": { "style": "bar", "blink": true },
  "sidebar": { "width": 200, "position": "left" },
  "theme": {
    "sidebar": { "background": "#000000" },
    "terminal": { "background": "#000000", "foreground": "#f8f8f2" }
  }
}

On Windows, the default shell is PowerShell (pwsh.exe if available, otherwise powershell.exe).

After editing, press Cmd+Shift+R (macOS) or Ctrl+Shift+R (Windows) to reload.

Building from Source

git clone https://github.com/clawterm/clawterm.git
cd clawterm
npm install
npm run tauri dev        # development
npm run tauri build      # production
npm run preflight        # lint, format, test, typecheck

Prerequisites: Rust (stable), Node.js (v18+), macOS or Windows. See CONTRIBUTING.md for platform-specific requirements.
